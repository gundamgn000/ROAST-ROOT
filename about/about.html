№ 001 / Roastery
// Our Story / 我們是誰

慢火
匆忙
烘焙。

Slow roast, honest daily coffee.

ROAST & ROOT 創立於 2024 年的台灣。
我們相信咖啡不是被催促出來的商品，而是一段從產地、烘焙到沖煮的日常節奏。
每週小批量烘焙，只提供我們願意每天喝的豆子。

// Manifesto / 咖啡主張

We don't rush flavor.
We roast with patience.

我們不追求快速，
我們相信慢火與時間。

// Beans. Fire. Water. Time. // 沒有過度包裝的話術，也不把咖啡講成難以靠近的學問。
我們只在意豆子的乾淨度、烘焙的穩定度，以及你每天願不願意再喝一杯。
好咖啡不該只存在於儀式感，也該存在於平凡早晨。

From Bean To Cup

// Our Principles 三個堅持。

01 / Principle

Fresh Roast.

新鮮烘焙

每週小批量烘焙，讓咖啡保留乾淨香氣與適合日常沖煮的穩定狀態。

02 / Principle

Clean Flavor.

乾淨風味

我們追求清楚、平衡、能被記住的風味，而不是只為了標籤或產區名稱而複雜。

03 / Principle

Daily Ritual.

日常儀式

咖啡不一定要嚴肅，它可以是工作前、午後或夜晚整理自己的小小節奏。

// Roast Process

From Bean

從生豆到日常杯中。

每一批咖啡都經過選豆、試烘、杯測、養豆與包裝。從產地風味到沖煮建議，我們盡量讓每一杯都穩定而清楚。

我們不追求大量出貨，而是保留小批量烘焙的彈性。因為真正舒服的風味，需要時間。
